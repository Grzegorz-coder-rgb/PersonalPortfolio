Backend Developer

Cena: 2800zł

Opanuj sztukę tworzenia solidnych i skalowalnych serwerów oraz API. Dowiedz się, jak zarządzać danymi i implementować złożoną logikę biznesową, która napędza nowoczesne aplikacje webowe. Kurs ten zapewni Ci głębokie zrozumienie technologii backendowych i najlepszych praktyk.

Program Kursu:

  • Wprowadzenie do programowania serwerowego i architektury klient-serwer
  • Podstawy języka Python w kontekście backendu
  • Framework Flask: tworzenie prostych API, routing, szablony Jinja
  • Framework Django: Model-View-Controller (MVC) / Model-View-Template (MVT), ORM, administracja
  • Zasady tworzenia RESTful API: metody HTTP, statusy, struktura odpowiedzi JSON
  • Bazy danych relacyjne (SQL): MySQL i PostgreSQL - projektowanie baz, zapytania SQL, indeksy, transakcje
  • Object-Relational Mapping (ORM): SQLAlchemy (dla Flaska), Django ORM
  • Bazy danych NoSQL (wprowadzenie do MongoDB): podstawy, zastosowania
  • Autoryzacja i autentykacja: sesje, tokeny JWT, OAuth2 (podstawy)
  • Zabezpieczenia aplikacji webowych: XSS, CSRF, SQL Injection, szyfrowanie haseł
  • Testowanie backendu: unit tests, integration tests
  • Deployment aplikacji backendowych: podstawy działania serwerów (nginx, Apache), konteneryzacja (Docker)
  • Zarządzanie zależnościami i środowiskami wirtualnymi
  • Asynchroniczność w Pythonie (asyncio - wprowadzenie)
  • Monitorowanie i logowanie aplikacji
  • Projekt końcowy: budowa kompleksowego API z bazą danych i autoryzacją.

Dla Kogo Jest Ten Kurs:

Kurs jest skierowany do osób, które chcą zbudować mocne fundamenty w programowaniu backendowym i nauczyć się tworzyć stabilne, bezpieczne i wydajne systemy. Idealny dla programistów Pythona chcących specjalizować się w web developmentcie po stronie serwera oraz dla tych, którzy chcą poszerzyć swoje umiejętności Full Stack o zaawansowany backend.

Czego Nauczysz Się na Kursie:

  • Projektować i implementować wydajne i bezpieczne API.
  • Efektywnie zarządzać bazami danych i manipulować danymi.
  • Wdrażać aplikacje serwerowe i zarządzać ich cyklem życia.
  • Zabezpieczać aplikacje przed typowymi zagrożeniami.
  • Testować kod backendowy, aby zapewnić jego stabilność.
  • Korzystać z popularnych frameworków Pythonowych (Flask, Django) do budowy backendu.
